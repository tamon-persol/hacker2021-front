# Dialog

## Usage
```jsx
import { Dialog } from 'components/atoms/Dialog'

<Dialog open={true}>
  <Box padding={4}>
    <h1>Title</h1>
    <p>TextTextTextTextTextTextText</p>
  </Box>
</Dialog>
```

## Props
Basically, [Material UI Dialog props](https://material-ui.com/ja/api/dialog/#props) are available.

## Note

`Dialog` is useing Material UI.
